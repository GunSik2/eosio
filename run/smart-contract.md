
## Wallet
### Create
```
cleos wallet create
```
### Open
```
cleos wallet unlock --password PW5JuBXoXJ8JHiCTXfXcYuJabjF9f9UNNqHJjqDVY7igVffe3pXub
```
### Lock
```
cleos wallet lock
```
### Import private key into wallet
```
cleos wallet import 5KQwrPbwdL6PhXujxW37FSSQZ1JiwsST4cqQzDeyXtP79zkvFD3
```
### List
```
cleos wallet list
```
### Keys
```
cleos wallet keys
```
